# Tabs Ongoing & Done

## Purpose
Memudahkan pengguna untuk melihat pekerjaan aktif vs pekerjaan yang telah selesai.

## Requirements
- Terdapat dua tab utama.
- Default tab adalah "Ongoing".

## UI Behavior
Ketika tab berubah, list task di bawahnya harus langsung terfilter sesuai status dari tab tersebut.
