# Task Management

## Purpose
Aplikasi ini ditujukan sebagai sarana pencatatan progres internal (Bugs, Adjust, Findings) secara sederhana dan cepat.

## User Flow
User masuk ke halaman utama -> Memilih tab Ongoing / Done -> Membaca, Membuat, Mengedit, atau Menghapus task.

## Requirements
- Tidak menggunakan autentikasi (bisa diakses semua tim internal).
- Semua operasi berada dalam satu halaman (`/`).

## Acceptance Criteria
- User dapat mengelola seluruh task pada satu interface sederhana.
