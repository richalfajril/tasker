# Task Status

## Purpose
Membagi task berdasarkan status pengerjaan agar mempermudah pemantauan.

## Requirements
Terdapat dua status:
1. **Ongoing**: Task yang masih dikerjakan (default).
2. **Done**: Task yang sudah selesai atau terimplementasi.

## UI Behavior
Task yang memiliki status `ongoing` akan berada di tab "Ongoing", sedangkan status `done` akan berada di tab "Done".

## Validation
Status yang diizinkan hanya `"ongoing"` dan `"done"`.
