# UI Layout

## Purpose
Mendefinisikan layout dasar halaman aplikasi.

## Requirements
- Header: Title, subtitle, dan tombol `+ New Task`.
- Container: maksimal lebar `1200px`, diletakkan di tengah (`mx-auto`).

## UI Behavior
- Single page layout.
- Responsif (mobile & desktop tetap menggunakan vertical list yang sama).
- Mendukung mode light dan dark.
